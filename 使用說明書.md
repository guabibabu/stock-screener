# 美股自動選股助手使用說明書

這是一個「輔助你篩選美股候選名單」的工具。它不會自動下單，也不是保證獲利工具。它的用途是幫你整理股票資料、打分數、排序，並用中文列出入選理由與風險提醒，方便你人工判斷。

## 1. 你可以用哪一版

目前有兩個版本可以用：

| 版本 | 適合誰 | 啟動方式 |
|---|---|---|
| 網頁版 | 推薦，一般使用最順 | 雙擊 `launch_us_stock_screener_web.command` |
| 桌面版 | 備用，傳統視窗介面 | 雙擊 `launch_us_stock_screener.command` |

建議先用網頁版。

網頁版啟動後，如果瀏覽器沒有自動打開，可以手動開：

```text
http://127.0.0.1:8765/
```

## 2. 最簡單使用流程

最簡單只要三步：

1. 打開網頁版。
2. 選擇策略模式。
3. 按「更新並篩選」。

如果你不想準備任何檔案，可以直接使用預設 ticker 清單。程式會自動連到 yfinance 抓資料，然後產生候選名單。

## 3. 是否需要網路

看你怎麼用：

| 使用方式 | 需要網路嗎 |
|---|---|
| 自動抓 yfinance 最新資料 | 需要 |
| 離線示範資料 | 不需要 |
| 上傳完整 snapshot 檔案 | 不需要 |
| 使用之前抓過的 snapshot | 不需要，但資料不是最新 |

如果你想每天抓最新可取得資料，就需要網路。

## 4. yfinance 抓的是什麼資料

按「更新並篩選」時，程式會透過 yfinance 從 Yahoo Finance 抓資料，包括：

- 股價
- 成交量
- 市值
- 公司基本資料
- 部分估值資料
- 可取得的財務欄位
- 歷史價格，用來計算波動、回撤、均線等

注意：

- yfinance 不是專業即時行情源。
- 它抓到的是 Yahoo Finance 當下提供的最新可取得資料。
- 價格可能接近即時，但可能延遲。
- 財報與基本面資料通常不是每天完整更新。
- 如果抓取失敗，報告會標示抓取失敗或資料缺失。

## 5. 策略模式

目前有兩種策略模式。

### Hybrid

日常混合篩選模式。

它同時看：

- 基本面
- 動量
- 風險
- 流動性

適合用來做每日或每週的候選股初篩。

### Stop Checking Price

偏長期、低頻率、品質導向的模式。

它比較重視：

- 公司品質
- 成長是否穩定
- 自由現金流
- ROE / ROIC
- 負債風險
- 股本稀釋
- 回撤控制
- 資料完整度

它不太鼓勵每天看短期股價波動，所以短線動量權重比較低。

## 6. 強制季度檢查是什麼

「強制季度檢查」只影響 Stop Checking Price 模式。

這個模式預設比較保守。平常不是正式季度檢查期時，即使股票分數很高，也通常只會放進觀察清單。

預設季度檢查期是：

- 3 月、6 月、9 月、12 月
- 且日期是該月 15 日之後

如果沒有勾選「強制季度檢查」：

- 可能看到 `WATCHLIST_HIGH_QUALITY`
- 意思是高品質觀察名單

如果勾選「強制季度檢查」：

- 程式會把今天當成正式季度檢查
- 高分且資料完整的股票可能顯示 `BUY_CANDIDATE`

它不會自動下單，只是改變報告裡的動作標籤。

## 7. 資料來源怎麼選

網頁版通常有三種資料來源：

### 使用預設 ticker 清單，自動抓 yfinance

最適合新手。

你不用準備檔案，直接按「更新並篩選」即可。

### 上傳自己的 CSV / JSON

適合你有自己的關注清單。

如果你的檔案只有 ticker，例如：

```text
ticker
AAPL
MSFT
NVDA
```

請勾選「自動抓 yfinance 資料」。

如果你的檔案已經有完整 snapshot，例如包含股價、市值、成交額等欄位，就可以不勾自動抓資料。

### 離線示範資料

適合測試介面是否正常。

它不會連網，也不代表你的真實股票結果。

## 8. 最低分數 min_score

最低分數是用來過濾結果的。

### Hybrid

如果你不填，預設顯示：

```text
min_score：未設定
```

意思是沒有額外設定最低分數門檻，只照排名顯示。

### Stop Checking Price

如果你不填，預設顯示：

```text
min_score：未設定
```

意思是先列出排名給你人工審查，不用固定 85 分把候選股藏起來。

如果你自己填分數，會以你填的為準。

## 9. 報告結果怎麼看

報告通常會包含：

### 候選排名

分數最高的股票排在前面。

你會看到：

- 排名
- Ticker
- 總分
- 動作標籤
- 入選理由
- 風險提醒

### 總分

分數越高，代表越符合目前策略的條件。

但分數不是買賣指令，只是排序輔助。

### 入選理由

例如：

- 營收與 EPS 維持正成長
- 營業利益率健康
- 自由現金流為正
- 回撤相對可控
- 長期趨勢尚未明顯轉弱

### 風險提醒

例如：

- 估值偏高
- 波動率偏高
- 回撤過深
- 負債偏高
- 資料完整度不足

### 硬性剔除

這代表股票在進入評分前就被排除。

常見原因：

- ETF
- ADR
- OTC
- 停牌
- 股價低於 5 美元
- 市值低於 20 億美元
- 20 日均成交額低於 2,000 萬美元
- 資料過舊

### soft penalty 扣分

這代表股票沒有被直接剔除，但有一些風險，所以被扣分。

常見原因：

- 自由現金流為負
- 營業利益率為負
- 負債偏高
- 股本稀釋偏高
- 估值偏高
- 回撤偏深
- 波動率極高

### missing data 缺資料

這代表資料不完整。

缺資料不一定代表公司不好，但代表你需要人工複查。

例如：

- ROIC 缺失
- free cash flow 缺失
- shares_growth_yoy 缺失

## 10. 常見動作標籤

| 標籤 | 意思 |
|---|---|
| `CANDIDATE` | 一般候選 |
| `WATCHLIST` | 放入觀察清單 |
| `WATCHLIST_HIGH_QUALITY` | 高品質觀察名單 |
| `BUY_CANDIDATE` | 季度檢查模式下的買入候選 |
| `HOLD_OR_REVIEW` | 可持有或進一步檢查 |
| `WATCHLIST_DATA_INSUFFICIENT` | 資料不足，只能先觀察 |
| `AVOID` | 暫時避免 |
| `EXCLUDE` | 被硬性剔除 |

這些不是自動交易指令。

## 11. 常見問題

### 按了更新很久沒完成

通常是 yfinance 或網路比較慢。

你可以看畫面上的執行狀態與已運行秒數。

### 為什麼很多股票被剔除

因為工具預設只適合大型 / 中大型、高流動性的美股普通股。

ETF、低流動性、小市值、OTC、ADR、資料缺失嚴重的股票會被排除或降分。

### 為什麼 Stop Checking Price 顯示很少候選

因為它比較保守，而且更重視品質、現金流、負債、資料完整度。若資料缺失嚴重，系統會降低信心分數或把動作限制在觀察清單。

### yfinance 抓不到怎麼辦

可能是：

- 沒網路
- Yahoo Finance 暫時連不上
- ticker 錯誤
- 該股票資料本來就不完整

可以稍後再試，或先用離線示範資料確認程式正常。

## 12. 建議使用方式

新手建議：

1. 先用網頁版。
2. 先用 Hybrid 看整體排名。
3. 再切 Stop Checking Price 看長期品質。
4. 不要只看總分，要看理由與風險。
5. 對缺資料的股票，一律人工複查。
6. 不要把任何結果當成保證買賣訊號。

## 13. 重要提醒

這個工具是投資決策輔助，不是投資建議，也不會保證結果。

你應該把它當成：

- 篩選工具
- 排序工具
- 風險提醒工具
- 人工研究前的第一層過濾

不要把它當成：

- 自動交易系統
- 保證獲利工具
- 唯一決策依據
